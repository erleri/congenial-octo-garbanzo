# PRM 시뮬레이터 (PRM Simulator)

브라우저 기반 Marimekko 차트를 활용한 매출/시장점유율/손익 최적화 시뮬레이터

## 🎯 주요 기능

- **Marimekko 차트 시각화**: 제품별 매출 구성을 직관적인 블록 형태로 표시
- **실시간 지표 계산**: 총 매출, 시장 점유율, 영업이익을 실시간으로 계산 및 표시
- **대화형 조정**: 슬라이더를 통해 제품별 시장 점유율, 가격, 원가를 조정
- **시장 환경 설정**: 총 시장 규모와 경쟁 강도를 설정하여 시뮬레이션
- **자동 최적화**: 이익률을 기준으로 최적의 제품 포트폴리오 자동 계산
- **최적화 제안**: 현재 설정에 대한 실시간 개선 제안 제공

## 🚀 사용 방법

1. `index.html` 파일을 웹 브라우저에서 열기:
   ```bash
   # 간단한 방법: 파일 탐색기에서 index.html을 더블클릭
   
   # 또는 로컬 서버 실행:
   python3 -m http.server 8000
   # 브라우저에서 http://localhost:8000/index.html 접속
   ```

2. **제품 설정 조정**:
   - 각 제품의 시장 점유율, 가격, 원가를 슬라이더로 조정
   - Marimekko 차트와 지표가 실시간으로 업데이트됨

3. **시장 환경 설정**:
   - 총 시장 규모 (억원)
   - 경쟁 강도 (1-10)

4. **자동 최적화 실행**:
   - "🚀 자동 최적화" 버튼을 클릭하여 이익률 기반 자동 최적화
   - 최적화 결과를 확인하고 필요시 수동 조정

5. **초기화**:
   - "🔄 초기화" 버튼으로 모든 설정을 초기값으로 복원

## 📊 지표 설명

- **총 매출**: 모든 제품의 매출 합계 (시장 규모 × 시장 점유율 × 가격)
- **시장 점유율**: 전체 제품의 시장 점유율 합계
- **영업이익**: 총 매출 - 총 원가 (경쟁 강도 영향 반영)
- **이익률**: 영업이익 / 총 매출 × 100%

## 🎨 기술 스택

- **HTML5**: 구조
- **CSS3**: 스타일링 및 그라데이션 효과
- **JavaScript (Vanilla)**: 비즈니스 로직 및 인터랙션
- **반응형 디자인**: 다양한 화면 크기 지원
